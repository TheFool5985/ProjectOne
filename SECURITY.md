## Reporting a Vulnerability

Please **do not** open public GitHub issues for security vulnerabilities.

Instead, report security issues privately by emailing:

**thefool5985@proton.me**

Include as much detail as possible:
- A description of the vulnerability and its impact
- Steps to reproduce (proof-of-concept is helpful)
- Affected components (frontend/backend), endpoints, versions, logs, etc.
- Any suggested mitigations or patches (if you have them)

### What to Expect
- You will receive an acknowledgement within a reasonable timeframe.
- We may ask follow-up questions to reproduce and validate the issue.
- If confirmed, we will work on a fix and coordinate disclosure timing with you.

## Disclosure Guidelines

We follow responsible disclosure practices:
- Please allow time for investigation and remediation before sharing details publicly.
- We may credit you in release notes unless you prefer to remain anonymous.

Thank you for helping keep this project and its users safe.
